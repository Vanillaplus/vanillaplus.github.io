To start you need to have UNMODIFIED WoW 1.12.1(5875) game client - Google it to find out how to get it.

Then you need to download our custom patch-3.mpq file and to drag it into your /WoW/Data/ folder and edit your Realmlist.wtf file "set realmlist logon.vanillaplus.org"

If you did it right, there will be an updated login screen when you'll launch the game.

If you are lazy, you can download game ready client (with patch-3 included)

Almost done! Create your account and enter the game!
